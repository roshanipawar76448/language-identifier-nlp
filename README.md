# Language Identifier using NLP

This project implements a simple language identification system using Natural Language Processing techniques.

## Technologies Used

* Python
* NLTK
* Jupyter Notebook

## Features

* Text preprocessing using NLP techniques
* Language prediction using n-gram probability models
* Visualization of prediction probabilities

## How to Run

1. Open the Jupyter Notebook file.
2. Install required libraries (NLTK, matplotlib if used).
3. Run the notebook and input text to detect the language.
